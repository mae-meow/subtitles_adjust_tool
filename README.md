# subtitles_adjust_tool
Simple JS + HTML that allows you to paste subtitles and shift all of them for given amount of seconds.

## Usage
[Demo version](https://mae-meow.github.io/subtitles_adjust_tool/). For running locally, clone this repo and then just open index file in the browser.
